# 版本管理文档

本文档记录 `cdu_teacher_jobs` 项目的版本变更历史，与 Git 提交记录互补。Git 记录代码维度，本文档记录项目维度的关键决策。

---

## 版本规范

采用语义化版本号：`主版本.次版本.修订号`

- **主版本 (Major)**: 架构重大变化，不兼容旧版
- **次版本 (Minor)**: 新增功能，向后兼容
- **修订号 (Patch)**: Bug修复、数据更新、配置调整

## v1.0.0 (2026-05-29)

### 项目初始化

**创建者:** 史超凡（成都大学教育学院研究生）

**目标:** 追踪成都23个区县的小学数学教师招聘信息，支持自动更新。

### 技术选型决策

1. **数据格式选 JSON 而非数据库** — 理由：
   - 数据量小（预计<500条/年），JSON 性能足够
   - 手动编辑友好，Git diff 可读
   - 零依赖，不需要额外安装数据库
   - 便于与 HTML 生成器集成

2. **爬虫选 Python + requests/BS4** — 理由：
   - Python 在爬虫领域生态最好
   - BeautifulSoup 容错性强（适合不规范的政府网站HTML）
   - 与 update_html.py 共享语言

3. **HTML 用纯字符串拼接而非模板引擎** — 理由：
   - 页面结构简单，模板引擎增加依赖
   - 纯 Python 字符串拼接可读性好
   - 零外部依赖，部署更简单

4. **定时任务用 Windows Task Scheduler** — 理由：
   - Windows 原生支持，不需要第三方工具
   - 系统级调度，开机自动恢复
   - PowerShell 安装脚本一次性配置

5. **Git 版本管理** — 理由：
   - 追踪代码变更
   - 数据 JSON 也在仓库中，可追溯历史招聘信息
   - 便于协作和备份

### 数据源设计

选择了5个数据源（3个官方+2个商业平台）：
- 成都市人社局 — 覆盖事业单位统一招聘
- 成都人才网 — 覆盖区县单独招聘
- 四川人事考试网 — 覆盖省级招聘
- 智联招聘 — 覆盖民办/培训机构
- 前程无忧 — 覆盖民办/培训机构

### 初始数据

手动录入了以下数据作为种子：
- 2025年：20条（含上半年统一招聘、夏季补招、下半年统一招聘、民办学校单独招聘）
- 2026年：21条（含已截止、进行中和即将开始的岗位）
- 研究生专区：10条（精选硕士友好岗位）

### 页面设计

- **主页面 (index.html)**: 年份Tab切换 + 区域分组表格 + 搜索/状态过滤
- **研究生专区 (grad.html)**: 硕士友好岗位精选 + 求职建议卡片
- **统一样式 (style.css)**: 响应式设计，移动端适配

## Git 分支策略

- `main` — 稳定版本，HTML 可直接部署
- `dev` — 开发分支，新功能测试
- `data-update` — 纯数据更新分支，用于爬虫自动提交

### 提交规范

```
feat: 新功能
fix: 修复
data: 数据更新
docs: 文档
refactor: 重构
style: 样式调整
```

示例：
```
feat: 增加研究生专区页面
data: 爬虫更新2026年6月招聘数据
fix: 修复锦江区区域识别错误
```
